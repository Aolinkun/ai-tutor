# AI Tutor · 项目说明文档

> 最后更新：2026-04-03
> 当前版本：v1.4.0

---

## 这是什么

基于 Benjamin Bloom Two Sigma 研究的苏格拉底式学习导师 Skill，适用于 Claude Code。

AI 不讲解——而是主动提问，根据你的回答实时调整难度，学习进度存成文件跨对话保存。

---

## 文件说明

| 文件 | 用途 |
|------|------|

## 安装方法

### 方法一：一键安装（推荐）

```bash
curl -fsSL https://raw.githubusercontent.com/Aolinkun/ai-tutor/main/install.sh | bash
```

### 方法二：从本地文件安装（GitHub 挂了或网络差时用）

```bash
mkdir -p ~/.claude/skills/ai-tutor/references
cp bloom-mastery-tutor/SKILL.md ~/.claude/skills/ai-tutor/
cp bloom-mastery-tutor/references/*.md ~/.claude/skills/ai-tutor/references/
```

### 卸载后重新安装

```bash
rm -rf ~/.claude/skills/ai-tutor
curl -fsSL https://raw.githubusercontent.com/Aolinkun/ai-tutor/main/install.sh | bash
```

## 使用方法

在 Claude Code 中说：

```
我想学 XXX              ← 开始新主题（会先问清楚目的再教）
继续学 XXX              ← 从上次进度继续
提示                    ← 卡壳时获得分级提示
跳过                    ← 跳过当前题，记录为薄弱点
```

---

## 学习进度存储位置

进度文件存在**你启动 Claude Code 的项目目录**下：

```
你的项目目录/
└── learning/
    └── [主题名称]/
        ├── progress.md        ← 进度、薄弱点、关键知识
        ├── unit-01-xxx.md
        ├── unit-02-xxx.md
        └── summary.md         ← 结课后生成
```

每次续学前确保在同一个目录下启动 Claude Code。
