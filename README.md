# AI Tutor · 苏格拉底学习导师

基于 Benjamin Bloom 的 **Two Sigma 掌握学习法**，让 Claude Code 变成你的专属一对一导师。

AI 不给你讲解——而是主动向你提问，根据你的回答实时调整难度。技能型主题还会布置实战任务，检验你真的会做，不只是说得出来。

---

## 一键安装

```bash
curl -fsSL https://raw.githubusercontent.com/Aolinkun/ai-tutor/main/install.sh | bash
```

需要：Claude Code，macOS / Linux

---

## 使用方法

在 Claude Code 中直接说：

```
我想学短视频文案创作   ← 技能型：掌握检测 + 实战任务
我想学经济学           ← 理解型：掌握检测
继续学短视频文案       ← 从上次进度继续
提示                   ← 卡壳时获得分级提示
跳过                   ← 跳过当前题，记录为薄弱点
```

---

## 核心机制

研究表明：一对一辅导 + 掌握学习法，可让学生超越 98% 的同龄人（Bloom, 1984）。

- **先澄清主题** — 「我想学抖音」→ 先问你想解决什么问题 → 收窄成精准主题
- **诊断再教学** — 找到你真实的知识边界，不瞎教
- **关键词检查** — 每个概念预设核心关键词，回答缺了关键词不能给✅
- **拒绝模糊表述** — 「好像」「大概」「差不多」自动降级，必须重新表述
- **掌握才推进** — 正确率不够退回巩固，不假装掌握了
- **理解型 vs 技能型** — 技能型主题额外布置实战任务，说得出来还不够
- **三级提示** — 卡住说「提示」，分级给线索，不直接给答案
- **情绪感知** — 连续答错自动降难度、换方式
- **每5单元复习** — 间隔重复，防止遗忘
- **上下文压缩** — 每3单元把关键信息存入文件，长对话不崩
- **跨主题关联** — 学新概念时主动关联你学过的其他主题

## 文件结构

安装位置：
```
~/.claude/skills/ai-tutor/
├── SKILL.md
└── references/
    ├── theory.md
    └── difficulty-levels.md
```

学习进度（存在你的项目目录）：
```
./learning/
└── [主题名称]/
    ├── progress.md
    ├── unit-01-[标题].md
    ├── unit-02-[标题].md
    └── summary.md              ← 结课后自动生成
```
